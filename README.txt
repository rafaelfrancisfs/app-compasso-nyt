
INSTRUÇÕES:
    - Para executar o projeto no smartphone é necessário usar o aplicativo EXPO;
    - Execute via terminal yarn start ou npm start;
    - Leia o QRCode com a camera de smartphone;
    - Abra via EXPO;

INFO:
    - utilzado o createAppContainer e createStackNavigator para ter um navegação mais simples;
    - utilzado o react hooks useEffect para carregar o fetch com as noticias;
    - link tipo TouchableOpacity para trazer uma navegabilidade mais fluída para que está usando;
    - o modal com webview para ter a experiência tipo SPA;
